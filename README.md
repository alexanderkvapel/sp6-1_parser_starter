# Парсер страницы

### Описание

Учебный проект 6.1.

Парсер страницы, который с помощью метода `parsePage()` возвращает информацию в виде объекта:
```json
    meta: {
        ...
    },
    product: {
        ...
    },
    suggested: [
        {
            ...
        }
    ],
    reviews: [
        {
            ...
        }
    ]
```

Тесты в `index.html` в строке `void makeTests(parsePage);`
